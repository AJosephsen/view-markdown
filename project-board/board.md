# view-markdown — Project Board

A single-file drop-in markdown viewer for the browser. No build step, no framework. Renders any `.md` via `?path=`. Hosted publicly at [ajosephsen.github.io/view-markdown](https://ajosephsen.github.io/view-markdown/).
